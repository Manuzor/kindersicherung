Kindersicherung
===============

Application that intercepts keyboard input and rejects most of the non-alphanumeric keys.

Build
=====

```zig
$ zig version
0.11.0
$ zig build install -Drelease -Dstrip
```

Can also use `build_release.bat` on windows to make a (proper ) release build.
